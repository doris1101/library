## Configure

```shell
	$ git config -global user.name 'yourname'
	$ git config -global user.email 'mailbox'
```

## Initialization item

```shell
	$ git clone 'http(s) or ssh'
	# or
	$ git init 'item_name'
```

## Push
```shell
	$ git add --all
	$ git commit -m 'comment'
	$ git push -u origin master
```

## More command
```shell
	$ git status
	$ git log
	$ git remote add 'remote_repo' 'url'
```
